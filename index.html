Description

This project demonstrates the development of a machine learning application that predicts building heights from Sentinel satellite images (Sentinel-1 and Sentinel-2). How it works

Client side contains different regions in Germany that can be selected from. Backend will process the selected region by getting the already preprocessed image file and further preprocess it. The processed image is then passed to the deep learning model for prediction and the predicted heights then send to frontend allowing the user to visualize and interpret the results effectively

Hybrid Website Analyzer

web development
❮ ❯

Project Details

Description

Hybrid Website Analyzer is a tool that inspects websites vulnerabilities from performance to security. The tool analyzes performance metrics such as Largest Contentful Paint (LCP), First Contentful Paint (FCP) amongst other metrics such as SI & CLS and offer suggestions on how to make a perfect score for your website. For network diagnosis it gets the ping time and trace route while for security it analyzes different vulnerabilities that can make you site hackable or leak information and categories the risks into different priorities. The tool utilizes bard AI to detect anomalies in the site hence offering real time evaluation of your site to make it safe and accessible to the intended group
